# Formation Python 🐍 — Du Débutant à l'Expert

Formation complète et progressive en programmation Python, conçue pour vous emmener du niveau débutant jusqu'à expert, avec une approche **projets progressifs**.

## Structure

La formation est découpée en **4 parties** et **36 modules**. Chaque module contient :
- `cours/` — cours en Markdown
- `exercices/` — scripts Python à compléter

## Comment utiliser cette formation

1. **Suivez l'ordre** — Chaque module construit sur le précédent
2. **Lisez le cours** dans le dossier `cours/`
3. **Faites les exercices** dans le dossier `exercices/`
4. **Validez avec les tests** fournis
5. **Passez au projet** associé en fin de module

## Progression recommandée

- **Partie 1** (modules 1-10) : ~2-3 semaines
- **Partie 2** (modules 11-20) : ~3-4 semaines
- **Partie 3** (modules 21-30) : ~4-5 semaines
- **Partie 4** (modules 31-36) : ~3-4 semaines

Bon apprentissage ! 🚀
